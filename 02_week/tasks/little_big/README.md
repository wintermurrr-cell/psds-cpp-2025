# Представление числа в памяти

Необходимо написать функцию `PrintMemory`, которая будет печатать битовое
представление числа в памяти. Функция выводит на экран битовое представление 
числа в шестнадцатеричном формате `0xYYYY` соответствующей размерности.

В конце вывода необходим перевод строки.

Одна версия функции должна принимать должна принимать число типа `int`, 
другая `double`. Вторым аргументом функция принимает флаг, который 
инвертирует порядок байт, по умолчанию порядок не инвертируется.

## Ограничения

- Запрещено использовать С-style cast и специализированные контейнеры.
