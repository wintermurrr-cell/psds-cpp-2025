# Фильтрация данных

Необходимо реализовать функцию `Filter`, которая принимает `std::vector<int>`
и указатель на функцию предикат, и оставляет в контейнере элементы, 
удовлетворяющие предикату, в исходном порядке. 

## Примечание

- Запрещено использовать функции из `<algorithm>`
- Алгоритм должен работать за линейное время `O(n)` (за один проход)
- Постарайтесь не выделять дополнительную память, дополнительная память `O(1)`
