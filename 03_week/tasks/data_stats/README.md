# Статистика данных

Необходимо реализовать функцию `CalculateDataStats`, которая принимает на вход 
контейнер `std::vector<int>` и возвращает структуру `DataStats`, содержащую 
среднее значение ([average]) и стандартное отклонение ([standard deviation]). 

## Примечание

- Запрещено использовать алгоритмы из `<algorithm>`
- Рекомендуется решить задачу в один проход по контейнеру
