# Уникальные элементы

Необходимо реализовать функцию `Unique`, которая принимает отсортированный
вектор `std::vector<int>` и возвращает вектор уникальных элементов в исходном 
порядке. 

## Примечание

- Запрещено использовать функции из `<algorithm>`
- Рекомендуется избегать частых реалокаций памяти
- Вместимость вектора должна соответствовать количеству элементов
