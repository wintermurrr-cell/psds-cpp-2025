# Диапазон элементов

Необходимо реализовать функцию `Range`, которая принимает на вход значение
начала диапазона, конца диапазона, шаг, и возвращает вектор `std::vector<int>` 
значений из диапазона `[from, to)` с шагом `step`. Шаг по умолчанию равен 1.

## Примечание

- Запрещено использовать функции из `<algorithm>`
- Рекомендуется избежать реалокаций памяти, то есть выделить память под нужное
  количество элементов заранее
